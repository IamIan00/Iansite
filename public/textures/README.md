# Earth Texture Maps

Download high-quality Earth textures for the 3D globe:

## Required Files

1. **earth_day_2k.jpg** (2048x1024)
   - Day-time Earth texture
   - Source: https://www.solarsystemscope.com/textures/ (free)
   - Or: https://planetpixelemporium.com/earth.html

2. **earth_night_2k.jpg** (2048x1024)
   - Night-time Earth texture with city lights
   - Source: https://www.solarsystemscope.com/textures/
   - Or: NASA Visible Earth

3. **earth_clouds_2k.png** (2048x1024)
   - Transparent cloud layer
   - Source: https://www.solarsystemscope.com/textures/

## Quick Setup

### Option 1: Solar System Scope (Recommended)
1. Visit https://www.solarsystemscope.com/textures/
2. Download "Earth" texture pack (free for non-commercial use)
3. Extract and rename files to match above names
4. Place in this directory

### Option 2: NASA Public Domain
1. Visit https://visibleearth.nasa.gov/
2. Search for "Earth" and download "Blue Marble" images
3. Resize to 2048x1024 if needed
4. Place in this directory
